# エールコネクト（Yell Connect）

> 応援（**Yell**）を、世代を超えてつなぐ（**Connect**）。
> 部活動の「現役生」と「OB/OG」をつなぐ、部活動支援プラットフォームの **MVPモックアップ**です。

社内提案・ピッチデモ用に作成した、**クリック可能なモックアップ**です。実際の決済・データベースは持たず、すべてダミーデータで動作します。

---

## ⚡ すぐに動かしたい方へ（3つの起動方法）

| 方法 | 必要なもの | 手順 |
| --- | --- | --- |
| **① GitHub Pages（公開URL）** | ブラウザのみ | 公開URLを開くだけ（常時アクセス可・インストール不要）。※公開設定後にURLをここに記載 |
| **② Node不要・静的HTML版**（手元で手軽に） | ブラウザのみ（任意でPython） | リポジトリ直下の [`index.html`](index.html) を**ダブルクリック**、または直下で `python -m http.server 8000` → http://localhost:8000/ |
| **③ Next.js版**（本来のコードベース） | Node.js | `npm install && npm run dev` → http://localhost:3000 （下記参照） |

> どれも画面・操作感は同等です。**配布・ピッチには ①、Node未導入の手元確認には ② が最短**です。
> `index.html` は外部通信ゼロの単一ファイル（オフライン動作）で、GitHub Pages の公開エントリも兼ねています。

## 🎯 このモックでできること

スマホUI（縦長カード）で、以下の2つの立場を**ワンタップで切り替え**ながら体験できます。

- **現役生として見る** … 活動を投稿し、欲しい物リストで支援を募る
- **OB/OGとして見る** … 後輩の活動を応援し、物品/金額で支援する

画面右上の「**現役生 / OB/OG**」トグルでいつでも切り替え可能です（選択状態は localStorage に保存されます）。

## 🚀 起動方法

### 前提：Node.js のインストール

この環境には Node.js が入っていないため、まず **Node.js 18.17 以上（推奨：LTS 20.x）** をインストールしてください。

- 公式サイト: https://nodejs.org/ja
- インストール後、ターミナルで確認：

```bash
node --version
npm --version
```

### セットアップ & 起動

プロジェクトのルート（この README がある階層）で、以下を実行します。

```bash
# 1. 依存パッケージのインストール
npm install

# 2. 開発サーバーの起動
npm run dev
```

起動後、ブラウザで **http://localhost:3000** を開いてください。

> モバイル前提のUIです。PCで見る場合はブラウザの**デバイスツールバー（スマホ表示）**を使うか、画面中央のスマホ枠の中で操作してください。

### その他のコマンド

```bash
npm run build   # 本番ビルド
npm run start   # 本番ビルドの起動
npm run lint    # Lint チェック
```

## 🧩 主な体験ポイント（デモの見どころ）

- **ロール切替**：上部トグルで現役生 ↔ OB/OG を即時切替。ホーム・フィード・サポートの表示が変化します。
- **ワンタップ支援**：欲しい物リストの「支援する」→ ダミー決済完了モーダル → 充足ゲージと支援履歴に即反映。
- **応援ランク**：支援額の年間集計に応じて 🥉ブロンズ / 🥈シルバー / 🥇ゴールド を自動判定。次ランクまでの進捗も表示。
- **いいね・応援コメント**：OBが現役生の投稿に反応 → カウントとコメントがその場で増えます。
- **観戦予定 / イベント参加**：ワンタップで予定登録・参加人数が増減。

> いいね・コメント・投稿・支援などの操作はすべて**メモリ上の状態**に反映されます。ページを再読み込みすると初期データに戻ります（ロール選択のみ localStorage に保持）。

## ⚠️ 注意事項（モックの制約）

- 実際の**決済・課金は行われません**（「デモ用のダミー決済」と明記したモーダルを表示）。
- **データベース・ログイン認証はありません**。ロールはトグルで切り替えるダミー方式です。
- 操作結果はリロードで初期化されます（ロール選択を除く）。
